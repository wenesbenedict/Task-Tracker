# Task Tracker

A MERN Stack Task Management Application built as part of an internship project.

## Features

- User Authentication (JWT)
- Login System
- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Mark Tasks as Failed
- Task History Tracking
- User Profile Settings
- MongoDB Database Integration

## Tech Stack

Frontend:
- React
- Vite
- CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas
- Mongoose
